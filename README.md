## How To Use

To clone and run this application, you'll need [Git](https://git-scm.com), [Go] installed on your computer. From your command line:

```bash
# Clone this repository
$ git clone https://github.com/marahaly/kasir-api.git

# Go into the repository
$ cd kasir-api

# Install Go

# Run the app
$ go run main.go
```
